## Installation

```bash
# Klona repot
git clone [repo-url]
cd [projektmapp]

# Installera beroenden
npm install

# Skapa .env (kopiera från .env.example)
cp .env.example .env
# Fyll i dina värden i .env

# Starta servern
node server.js
# eller med --watch:
node --watch server.js
```

Servern startar på `http://localhost:3000` (eller den port du angett i `.env`).
